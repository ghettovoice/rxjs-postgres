# NOT READY FOR PRODUCTION

## System requirements

* PostgreSQL ver. >= 9.1

## Installation

```bash
npm install --save rxjs-postgres
```

## Testing

```bash
# Run all
npm test

# Run unit tests
npm run test:unit

# Run integration tests
npm run test:integration
```
