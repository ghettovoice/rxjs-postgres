## System requirements

* PostgreSQL ver. >= 9.1

## Installation

```bash
npm install --save rxjs-postgres
```

## Package commands

```bash
# run tests
npm test

# compile sources
npm run compile

# generate documentation
npm run doc
# deploy documentation to GitHub Pages
npm run doc:deploy

# lint sources
npm run lint
```
